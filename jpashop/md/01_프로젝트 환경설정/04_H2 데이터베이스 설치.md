### H2 데이터베이스 설치
- 개발이나 테스트 용도로 가볍고 편리한 DB, 웹 어드민 화면 제공

*설치 및 실행*
- 설치 시 Version 1.4.200 사용할 것
- 설치 후 h2 > bin 폴더로 이동
  - `./h2.sh` 명령어 실행
- 실행 후 h2 database 웹 페이지에서 
    - `jdbc:h2:~/jpashop` 로 최초 실행 -> 파일 모드로 접근
    - `~/jpashop.mv.db` 파일 생성 확인
    - `jdbc:h2:tcp://localhost/~/jpashop` 네트워크 모드로 다시 실행
