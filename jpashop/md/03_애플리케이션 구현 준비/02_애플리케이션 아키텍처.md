### 애플리케이션 아키텍처

<img src="애플리케이션%20아키텍처.png" width="60%"/>

*계층형 구조 사용*
- Controller, Web: 웹 계층 
- Service: 핵심 비즈니스 로직, 트랜잭션 처리
- Repository: JPA 를 직접 사용하는 계층, 엔티티 매니저 사용
- Domain: 엔티티가 모여 있는 계층, 모든 계층에서 사용
    
*패키지 구조*
- jpabook.jpashop
    - domain
    - exception
    - repository
    - service
    - web
